129/* ============================================================================
130 * SECTION PLACEMENT MACROS
131 *
132 * These macros annotate functions and data with their intended memory region.
133 *
134 * Memory region mapping:
135 * ITCM_FUNC → .itcm (hot-path code: handlers, wrappers, context switch)
136 * DTCM_DATA_PRIV → .dtcm_priv (privileged-only kernel data: task_list,
137 * semaphore_list, scheduler state)
138 *
139 * MPU protection:
140 * - ITCM (.itcm): Read-only + Execute for all (Region 0)
141 * * Prevents code modification attacks
142 * * Kernel handlers protected by ARM exception architecture
143 * - DTCM (.dtcm_priv): Privileged-only RW (Region 5)
144 * * Kernel data isolated from unprivileged tasks
145 * * Access via SVC call gates only
146 * ========================================================================= */
147
