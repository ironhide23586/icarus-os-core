Detailed Description

ICARUS OS Task Management Implementation.

Version
0.1.0

Core kernel implementation including:

  • Task creation and lifecycle management
  • Round-robin preemptive scheduler
  • Counting semaphores
  • Message pipes (queues)
Memory Layout (STM32H750):
  • DTCM (128KB): Task pools, semaphores, pipes - zero wait-state
  • ITCM (64KB): Hot-path functions - zero wait-state
  • RAM_D1 (512KB): Stack pool, BSS
See also
docs/do178c/design/SDD.md for detailed design documentation
STM32H750 Reference Manual RM0433 for memory architecture
Author
Souham Biswas
Date
2026

◆ os_yield()

void os_yield ( void )

Voluntarily yield CPU to scheduler.

Triggers a context switch to the next ready task. The current task is placed in READY state and will be scheduled again.

Note
This function is placed in ITCM for zero wait-state execution.
Implementation:
Sets PendSV pending bit to trigger context switch at lowest interrupt priority, ensuring clean stack state.
See also
ARMv7-M Architecture Reference Manual, Section B1.5.14 (PendSV)
