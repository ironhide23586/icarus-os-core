Detailed Description

Board Support Package - Terminal Display Driver.

Version
0.1.0

ANSI terminal-based display for task visualization. Renders progress bars, status indicators, and message queues over USB CDC serial connection.

Terminal Requirements:
  • ANSI escape code support
  • UTF-8 encoding (for box drawing characters)
  • Minimum 80x24 terminal size recommended
Author
Souham Biswas
Date
2026

Initialize terminal display.

Initialize terminal display.

Performs one-time initialization of the terminal display:

  • Renders ICARUS ASCII art logo header

Precondition

Postcondition
Terminal ready for real-time updates

This function is idempotent - subsequent calls are no-ops
