Detailed Description

Board Support Package - USB CDC Driver.

Version
0.1.0

USB Full-Speed CDC (Virtual COM Port) interface. Used for printf() output and debug console.

USB Configuration:
  • Mode: Device
  • Class: CDC (Communications Device Class)
  • Speed: Full-Speed (12 Mbps)
  • VID/PID: 0x0483/0x5740 (STMicroelectronics)
