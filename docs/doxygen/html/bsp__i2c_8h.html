Detailed Description

Board Support Package - I2C Driver.

Version
0.1.0

I2C2 driver for IMU (LSM9DS1) interface.

I2C2 Configuration:
  • Speed: 400kHz (Fast mode)
  • Address mode: 7-bit
  • Pins: PB10 (SCL), PB11 (SDA)
See also
STM32H750 Reference Manual RM0433, Section 52 (I2C)
LSM9DS1 Datasheet for register map

Souham Biswas
