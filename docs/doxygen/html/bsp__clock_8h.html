Detailed Description

Board Support Package - Clock Configuration.

Version
0.1.0

System clock, PLL, and power management configuration.

Clock Tree (STM32H750 @ 480MHz):
HSE (25MHz) -> PLL1 -> SYSCLK (480MHz) -> HCLK (240MHz) -> APB1 (120MHz) -> APB2 (120MHz)
See also
STM32H750 Reference Manual RM0433, Section 8 (RCC)

void bsp_clock_init ( void )

Configure system clocks.

Configures:

  • HSE oscillator (25MHz external crystal)
  • PLL1 for 480MHz SYSCLK
  • AHB/APB prescalers
  • Flash latency
Note
Called by bsp_init()
